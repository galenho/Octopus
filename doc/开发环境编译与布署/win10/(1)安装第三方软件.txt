1. 安装配置mysql数据库

		第1步: 安装数据库
	//----------------------------------------------------------------

	mysql-installer-community-5.7.28.0.exe
	
	//----------------------------------------------------------------
		第2步: 修改/etc/my.cnf文件，编辑配置文件如下
	//----------------------------------------------------------------

	找到character-set-server, 修改为以下内容：

	[mysqld]
	character-set-server = utf8
	
	//----------------------------------------------------------------

	//----------------------------------------------------------------
	mysql -uroot -p
	输入密码: 88104725galenho

	mysql>
	grant ALL PRIVILEGES on *.* to root@'127.0.0.1' Identified by '88104725galenho' with grant option;
	grant ALL PRIVILEGES on *.* to root@'localhost' Identified by '88104725galenho' with grant option;
	grant ALL PRIVILEGES on *.* to root@'%' Identified by '88104725galenho' with grant option;
	grant ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;
	FLUSH PRIVILEGES;
	EXIT
	

2. 安装配置mongo 4.0.0数据库 

	//----------------------------------------------------------------
		第1步: 安装数据库
	//----------------------------------------------------------------
	
	mongodb-win32-x86_64-2008plus-ssl-4.0.0-signed.exe

		注：mongodb Windows系统下安装卡死问题

			原因分析：原因在于，默认安装是包含Mongodb compass（就是一个可视化操作MongoDB数据库的操作界面）,这个包的，这个包大约有180MB，下载很慢，导致所谓的的”卡死“。
			解决方法：选用自定义安装的并且不安装Mongodb compass（前期不安装也没有任何影响），安装就非常容易了，如果你想使用Mongodb compass，可以从别的网站上单独下一个，也是可以使用的。

	//----------------------------------------------------------------
		第2步: 设置系统变量
	//----------------------------------------------------------------
	设置系统变量Path, 添加C:\Program Files\MongoDB\Server\4.0\bin


	//----------------------------------------------------------------
		第3步: 创建root用户并设置密码
	//----------------------------------------------------------------
	cmd ----> mongo ----> use admin ----> db.createUser( {user: "root",pwd: "88104725galenho",roles: [ { role: "root", db: "admin"} ]}) ----> exit

	//----------------------------------------------------------------
		第4步: 创建root用户并设置密码
	//----------------------------------------------------------------
	编辑bin\mongod.conf文件， 把bindIp: 127.0.0.1 改为 bindIp: 0.0.0.0


	注销一下系统

	cmd
	mongo -u "root" --authenticationDatabase "admin" -p "88104725galenho"
